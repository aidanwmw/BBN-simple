# BBN-simple

This repository contains the code for our paper [here](https://arxiv.org/abs/2412.07893) "BBN Simple: How to Bake a Universe Sized Cake." This repository contains a self-contained code using the reaclib strong rate fits for a simple and pedagogical big bang nucleosyntehsis code.

The jupyter notebook is hosted as an html file [here](aidanwmw.github.io).
